# AI Email Reply Assistant

An AI-powered email reply assistant built using React, FastAPI, and Groq AI.  
This application helps users generate professional, context-aware email replies instantly.

---

## Features

- AI-generated professional email replies
- Personalized responses using user's name
- Additional context support for smarter replies
- Modern pastel green UI
- FastAPI backend
- React frontend
- Groq AI integration
- Responsive and user-friendly design

---

## Tech Stack

### Frontend
- React.js
- JavaScript
- CSS

### Backend
- FastAPI
- Python

### AI Integration
- Groq API
- Llama 3.3 70B Versatile
